# Change Log

All notable changes to the "low-gravitas-zen-theme" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-03-14

### Added

- New color tokens for VS Code features added since mid-2024: tab selected states, activity bar top layout, SCM graph, git blame annotations, ghost text, inline chat/edit, chat panel, sticky scroll (editor/sidebar/panel/terminal), diff editor unchanged regions, multi-diff editor, terminal command guide, and more
- Rainbow indent guides and bracket pair coloring

### Changed

- Unified terminal ANSI palette across all themes
- Alpha transparency on merge, minimap, and inactive selection overlays
- Visible yellow active tab border

## [0.1.0] - 2022-01-11

- Initial release
