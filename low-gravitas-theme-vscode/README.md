# Low Gravitas

A high-contrast, low-blue-light color theme for VS Code — in dark and light variants.

## Variants

- **Low Gravitas** — warm dark theme with cream/gold tones on deep brown-black backgrounds
- **Low Gravitas Light** — warm light theme with parchment backgrounds and darkened syntax colors

Both variants are designed to reduce blue light in large background areas while keeping syntax colors vivid and readable.

## Installation

Search for **Low Gravitas** in the VS Code Extensions panel, or install from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=low-gravitas.low-gravitas-theme).

After installing, open **Settings > Color Theme** and choose either variant.
